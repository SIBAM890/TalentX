## 🎯 Key Talking Points for Q&A

- **Why Sentence-BERT?** Free, private, fast, 384-dim semantic embeddings
- **Why 5 dimensions?** Each captures a different axis of fit that keywords miss
- **Why DishaSetu Mode?** India has 20M students making career decisions annually with no AI support
- **Scalability?** Batch embedding handles 1000+ resumes; Docker-ready for production
- **What's next?** Hindi language support, PDF parsing, LLM integration, LinkedIn API
